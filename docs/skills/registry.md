\# Registry Skill



\## Purpose



Maintain a history of approved infrastructure review reports.



\## Responsibilities



Store



\- Timestamp

\- Weather summary

\- Infrastructure findings

\- Evidence

\- Human decision

\- Reviewer comments



Retrieve previous reports.



\## Inputs



Approved Review



\## Outputs



Stored Report



\## Constraints



Only approved reports are stored.



\## Success Criteria



Reports can be retrieved chronologically.



\## Failure Behavior



If the report cannot be stored:



\- Notify the user.

\- Keep the report available in memory until it can be saved.
