\# Deployment Skill



\## Purpose



Deploy the application as a publicly accessible web service.



\## Responsibilities



Containerize the application.



Deploy to Google Cloud Run.



Configure environment variables.



Provide HTTPS endpoint.



\## Inputs



Application source code



\## Outputs



Public URL



Docker image



Deployment documentation



\## Constraints



API keys must never be stored in source code.



\## Success Criteria



The application is accessible from a browser.
